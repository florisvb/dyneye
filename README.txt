This is a ROS package used to generate and analyze all the data in the paper "Monocular distance estimation from optic flow during active landing maneuvers" by Floris van Breugel, Kristi Morgansen, and Michael Dickinson.

The code depends on several other packages that are either freely available on the internet, or freely available within this git repository: https://github.com/florisvb

The raw image files are available upon request (florisvb@gmail.com).
